Display A Map || Search Input Autocomplete and Geocoder || Placing Markers || Directions

In order to use the Google Maps Autocomplete search function you need to implement the places library. In your script <script async defer src="https://maps.googleapis.com/maps/api/js?key=YOUR_API_KEY&callback=initMap" type="text/javascript"></script> after your api key you need to add &libraries=places to the url. So your script url should look like this <script async defer src="https://maps.googleapis.com/maps/api/js?key=YOUR_API_KEY&libraries=places&callback=initMap" type="text/javascript"></script> Now that we took care of that, adding the Autocomplete search is super easy. At the end of your initMap() function put this line of code: 

autoSearch = new google.maps.places.Autocomplete(document.getElementById('search'));
Boom. That's it. Now we need to get the information from the search. Pseudocoding this, the user will type something in the search text field, google will autocomplete it, and when they hit the submit button, we need to get the value in the search text field and convert the location to latitude and longitude. For that, we will need the Geocoder. The google Geocoder object allows you to convert locations into geo coordinates, and geo coordinates into locations (that's called reverse geocoding). Add this line of code after your autoSearch variable 
geocoder = new google.maps.Geocoder();
So the geocoder gets initialized when the map gets initialized. When the user hits submit, we send the Geocoder the address, and it gives us coordinates. Put this code right under your initMap() function. Make sure to read the comments. Search a place then hit submit. Check your console for the coordinates to appear. If it works then AWESOME.
