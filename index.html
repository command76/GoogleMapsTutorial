The first thing you want to do is get an API key. Go to this link Get Javascript API Key. Click the Get A Key button and make sure to log in to your google account. Once you get a key, you can use this script <script async defer src="https://maps.googleapis.com/maps/api/js?key=YOUR_API_KEY&callback=initMap" type="text/javascript"></script>. Paste your API key where it says YOUR_API_KEY. Go ahead and paste this script above your </body> tag. Now we can start playing with google maps. Take a look at the script again and see the callback here: <script async defer src="https://maps.googleapis.com/maps/api/js?key=YOUR_API_KEY&callback=initMap" type="text/javascript"></script>. This callback is a function that runs when google maps is done loading. We use this function to display a map. They already have callback=initMap, so lets name our function initMap(). Right above our google maps script tag lets make another script tag with this:

So the initMap() function gets called when the API is done loading. The initMap() function first gets the div with id="map" and sets it to a variable called mapDiv. The map variable then creates a new google.maps.Map() object, the first parameter being mapDiv, where the map will be shown, and the second parameter are the map options placed as an object. To initialize a map you must have the center and zoom set in the options parameter. You can put other options in that object as well.
